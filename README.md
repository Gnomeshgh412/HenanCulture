## 1. 核心目标（一句话版）

### 从河南省及若干地市的地方志 / 民俗志文本中，自动抽取节日、婚俗、信仰等民俗要素，构建结构化的民俗数据库和简单知识图谱，并做空间/类型的统计分析与可视化。

## 2. 数据范围（一期工程）

### 省级：

《河南省志》中涉及民俗、文化、民间信仰的章节（具体卷名以后再细化）

### 市级（先选 1 个试点）：

例如：洛阳市志 或 开封市志 中的相关章节

### 文本形式：

优先：可复制的网页 / TXT / DOC / PDF（能转成文本）

后续再考虑更多市县扩展

## 3. 主要任务模块（技术视角）

### 数据采集与清洗

手动或半自动抓取地方志文本

去除页眉页脚、目录号、无关符号

### 文本预处理 & 分词

分句、分词、去停用词

### 民俗要素抽取

先规则 + 词典 + 关键词

后面有精力再上简易 NLP / 分类 / 序列标注

### 结构化存储

把“节日 / 婚俗 / 信仰”等整理成表格（CSV / SQLite 等）

### 可视化 & 基本分析

例如：

不同市县的节日频次热力图

民俗类别分布柱状图

### 知识图谱 Demo（可选进阶）

构建实体：地区、节日、人物、场所、仪式

构建关系：某地-举行-某节日、某节日-包含-某仪式
